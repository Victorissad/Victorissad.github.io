Contexte

Structur'All Machine — modernisation du code legacy

Structur'All Machine est la plateforme principale de MTB111 by Creative — une solution labellisée FinTech par Finance Innovation permettant aux institutions financières (BNP Paribas, Allianz, Banque de France) de moderniser leur code legacy sans réécriture complète. L'outil analyse, transforme et migre automatiquement du code source dans différents langages, avec une interface disponible à la fois en version locale et en version web.

Ma mission : valider la fiabilité et la cohérence des transformations produites par la plateforme sur des programmes de référence, en comparant les résultats obtenus en local et via l'interface web.

Périmètre

333 cas de test — 4 langages

JavaProgrammes de référence couvrant les structures de contrôle, classes, héritage, exceptions
C#Équivalents C# des cas Java, propriétés, namespaces, delegates
COBOLProgrammes COBOL représentatifs du code legacy bancaire (divisions, sections, paragraphes)
PHPScripts PHP couvrant les fonctions, classes, sessions, requêtes HTTP
Méthodologie

Déroulement des campagnes de tests

  • Rédaction des cahiers de recette — création des cas de test dans Excel : description, données d'entrée, résultat attendu, statut
  • Exécution local vs web — chaque programme de référence est soumis à Structur'All Machine en version locale puis en version web, et les sorties sont comparées
  • Détection d'anomalies — identification des divergences, erreurs de transformation ou comportements inattendus
  • Remontée via tickets — signalement structuré de chaque anomalie via système de ticketing interne, avec description précise et étapes de reproduction
  • Suivi des corrections — vérification après correction par l'équipe R&D, mise à jour du statut dans les cahiers de recette
